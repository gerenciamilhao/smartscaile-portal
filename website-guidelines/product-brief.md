# smartscaile. Portal — Product Brief

## Produto
Portal de propostas e diagnosticos para prospects da smartscaile. Acesso via token compartilhado por WhatsApp.

## Objetivo
Converter prospects em clientes atraves de uma experiencia visual premium pos-reuniao. O portal entrega valor tangivel (diagnostico + proposta) de forma que nenhuma agencia no mercado faz.

## Publico
Prospects que fizeram reuniao com a smartscaile. Perfil: gestores de trafego, infoprodutores, donos de e-commerce que investem em ads e precisam de tracking/analytics profissional.

## Fluxo do Usuario
1. Recebe link + token via WhatsApp
2. Acessa portal.smartscaile.com
3. Insere token no campo de acesso
4. Visualiza proposta/diagnostico personalizado com dados reais da reuniao
5. Clica no CTA para agendar implementacao
